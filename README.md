# De Roma — موقع المطعم

موقع مطعم دي روما (بيتزا ولازانيا إيطالية، بنغازي) مبني بـ Next.js + Tailwind + Supabase.

## التشغيل محليًا
```bash
npm install
npm run dev
```

## متغيرات البيئة
انسخ `.env.example` إلى `.env.local` وحط فيها بيانات Supabase الخاصة بمشروعك.
