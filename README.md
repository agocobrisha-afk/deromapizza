# De Roma — موقع وسكربت مطاعم بيتزا

موقع مبني بـ Next.js + Tailwind + Supabase، كل المحتوى (النصوص، الصور، المنيو، الصفحات) قابل للتعديل من لوحة تحكم بدون لمس الكود.

## التشغيل محليًا
```bash
npm install
npm run dev
```

## متغيرات البيئة
انسخ `.env.example` إلى `.env.local` وحط فيها بيانات Supabase.

## قاعدة البيانات
شغّل بالترتيب في Supabase → SQL Editor:
1. `supabase/migration-01-init.sql` (لو ما شغلتها قبل)
2. `supabase/migration-02-cms.sql`

## لوحة التحكم
الرابط: `/admin/login`
تحتاج تسوي مستخدم من Supabase Dashboard → Authentication → Add user.

## الحالة الحالية
- ✅ الصفحة الرئيسية (تصميم فاتح احترافي)
- ✅ صفحة المنيو الكامل (مربوطة بقاعدة البيانات)
- ✅ صفحات فرعية ديناميكية (`/[slug]`)
- ✅ نظام تقييمات الزبائن
- ✅ لوحة تحكم: إعدادات الموقع، المنيو، الصفحات، التقييمات
- ⏳ نظام السلة والطلبات المباشر من الموقع (قيد التطوير)
- ⏳ معرض ثيمات متعددة للصفحة الرئيسية (قيد التطوير)
